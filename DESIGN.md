# ILSI Europe Design System

## Brand Context

ILSI Europe is a global, nonprofit federation committed to improving public and planetary health worldwide through evidence-based scientific research on nutrition, food safety, and sustainability. Our design system reflects credibility, scientific integrity, collaboration, and pioneering thought leadership.

**Brand Tagline:** Collaborative science for safe, nutritious and sustainable food.

## Typography

### Primary Typeface

**Montserrat** — A modern, versatile sans-serif font designed by Julieta Ulanovsky
- Available free from Google Fonts
- Optimized for both digital and print applications
- Features: tall x-height for excellent legibility, extensive glyph set, multiple weights

### Font Weights

- **Light** — Used for secondary information and subtle hierarchy
- **Regular** — Body text and standard content
- **Medium** — Subheadings and emphasized content
- **Bold** — Headlines, primary headings, and strong emphasis

### Fallback Fonts

In instances where Montserrat cannot be used:
- First fallback: Helvetica
- Second fallback: Arial

## Logo

### Primary Logo

The ILSI logo consists of:
- **Icon:** Circular gradient mark with four colors (Indigo, Orange, Blue, Green) representing the tripartite sectors of academia, public health, and industry working in harmony
- **Wordmark:** "ILSI" in ILSI Indigo using Montserrat Bold

### Logo Variations

- **Color Version:** Full color logo for use on light backgrounds
- **Reverse Version:** White logo for use on dark backgrounds
- **Icon Only:** Logo icon alone when brand name is visible nearby
- **One Color:** Black version for monochrome applications

### Logo Usage Guidelines

- Maintain minimum clear space equal to the height of the ILSI wordmark on all sides
- Never stretch, distort, or alter the logo proportions
- Always use approved logo files
- Logo should never appear smaller than 1 inch in print or 100px on digital

## Brand Personality

ILSI Europe embodies eight core personality traits:

1. **Credible** — Trustworthy authority in scientific research
2. **Honest** — Transparent and ethical in all communications
3. **Collaborative** — Working across sectors and disciplines

5. **Smart** — Intellectually rigorous and sophisticated
6. **Crisp** — Clear, concise, and well-organized
7. **Open** — Inclusive and accessible

## Design Components

### Buttons

**Primary Button**
- Background: ILSI Indigo (#1F0079)
- Text: White
- Padding: 12px 24px
- Border Radius: 4px
- Font Weight: Medium (Montserrat Medium)
- Hover State: Darker shade or ILSI Blue

**Secondary Button**
- Border: 2px ILSI Orange (#F79F1E)
- Background: Transparent
- Text: ILSI Indigo
- Padding: 10px 22px
- Border Radius: 4px
- Font Weight: Medium

### Cards

- Background: White
- Border Radius: 8px
- Box Shadow: Light shadow (0 2px 8px rgba(31, 0, 121, 0.1))
- Padding: 24px
- Border: 1px solid #E0E0E0

### Headlines

- Font Family: Montserrat
- Font Weight: Bold
- Color: ILSI Indigo

**H1** — 32px, Bold
**H2** — 24px, Bold
**H3** — 20px, Medium

### Body Text

- Font Family: Montserrat
- Font Weight: Regular
- Size: 16px
- Line Height: 1.5
- Color: ILSI Charcoal (#393939)

### Links

- Color: ILSI Blue (#1B7AB2)
- Hover State: ILSI Indigo
- Text Decoration: Underline on hover

### Dividers

- Color: ILSI Orange (#F79F1E)
- Height: 2px
- Used to separate sections and create visual hierarchy

## Imagery & Photography

- Use photography that reflects sustainability, science, and collaboration
- High-quality images showing: food, nutrition, agriculture, scientific research
- Maintain consistent color temperature and style
- Overlay with brand colors for visual coherence

---

## Spacing & Layout

- Use multiples of 8px for consistent spacing (8px, 16px, 24px, 32px, 40px, 48px, etc.)
- Maintain generous white space for clarity and sophistication
- Organize content with clear visual hierarchy using font size and weight contrast

---

## Accessibility

- Ensure minimum contrast ratio of 4.5:1 for WCAG AA compliance
- All interactive elements must be keyboard accessible
- Use semantic HTML and ARIA labels
- Provide alt text for all images
- Use color plus additional visual indicators (not color alone)
