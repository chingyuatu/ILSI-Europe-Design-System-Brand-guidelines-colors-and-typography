# ILSI Europe Logo Specifications

## Logo Overview

The ILSI Europe logo is a carefully designed mark representing our commitment to collaborative science across three sectors: academic, public, and private. The identity consists of a distinctive circular icon paired with the ILSI wordmark.

---

## Logo Components

### Icon

**Design Elements**
- Circular gradient mark representing global collaboration
- Four overlapping color segments representing harmony across sectors
- **Primary Colors Used:**
  - ILSI Magenta (top-left) — #9C037E
  - ILSI Orange (top-right) — #F79F1E
  - ILSI Green (bottom-left) — #6CC185
  - ILSI Blue (bottom-right) — #1B7AB2

**Symbolism**
- The circular form represents global and international focus
- Color overlays emphasize working together and transparency
- The mandala-like design suggests security and sustainability
- Four quadrants represent the integration of different sectors

### Wordmark

**Typography**
- Font: Montserrat Bold
- Text: "ILSI"
- Subtitle: "EUROPE"
- Primary Color: ILSI Indigo (#1F0079) for standard use

## Logo Variations

### 1. Primary Logo (Color)

**Usage**
- Light backgrounds (white, cream, light gray)
- Digital applications (web, screens)
- Print materials on light substrates
- Primary default logo

**Specifications**
- Icon: Full color gradient
- Wordmark: ILSI Indigo
- Minimum size: 100px (digital) / 1 inch (print)
- File formats: PNG (transparent), SVG, PDF

### 2. Reverse Logo (White/Monochrome)

**Usage**
- Dark backgrounds (navy, charcoal, dark images)
- Dark-themed digital applications
- Print materials on dark substrates
- High-contrast applications

**Specifications**
- Icon: White with subtle outline
- Wordmark: White
- Minimum size: 100px (digital) / 1 inch (print)
- File formats: PNG (transparent), SVG, PDF

### 3. Icon Only

**Usage**
- Favicon, app icon, social media avatars
- When brand name is visible nearby
- Compact applications with space constraints
- Secondary brand touchpoints

**Specifications**
- Just the circular mark, no wordmark
- Maintain color or reverse version consistently
- Minimum size: 64px (digital)
- File formats: PNG, SVG, ICO, ICNS

### 4. One Color (Monochrome)

**Usage**
- Black and white printing
- Fax, photocopying, low-color reproduction
- Engraving, embossing
- Emergency fallback

**Specifications**
- Icon: Solid ILSI Indigo or Charcoal
- Wordmark: Same color
- File formats: EPS, PDF

## Clear Space (Minimum Spacing)

The logo requires breathing room to maintain impact and legibility.

**Rule**
The minimum clear space equals the **height of the ILSI wordmark** on all four sides of the logo.

**Application**
- Measure the height of the "ILSI" text
- Apply that distance as minimum spacing on all sides
- Do not place other design elements within this space
- More space is always acceptable

**Example**
- If "ILSI" is 1 inch tall
- Minimum clear space = 1 inch on top, bottom, left, right
- Total minimum logo footprint = 3 inches (logo width + 2 × clear space)

## Logo Sizing

### Minimum Sizes

| Medium | Minimum Size | Rationale |
|--------|--------------|-----------|
| Digital (web) | 100px width | Ensures icon legibility at standard screen resolutions |
| Mobile (app icon) | 64px width | Minimum OS requirement for app icons |
| Print (standard) | 1 inch width | Industry standard for print legibility |
| Print (small) | 0.5 inch | Minimum acceptable size; clarity may suffer |

### Scaling Guidelines

- **Always scale proportionally** — use constrained aspect ratio
- **Maintain aspect ratio** of approximately 4:1 (icon + wordmark)
- **Never distort, stretch, or squeeze** the logo
- **Responsive sizing** — adjust proportionally for different media

## Background Applications

### Color Logo

**Acceptable Backgrounds**
- Light solid colors (white, cream, light gray, pastels)
- Light photography with sufficient contrast
- Light gradient backgrounds

**Unacceptable Backgrounds**
- Dark backgrounds — use reverse logo instead
- Busy, high-contrast patterns
- Backgrounds with colors too similar to logo colors
- Transparent backgrounds without defined space

### Reverse/White Logo

**Acceptable Backgrounds**
- Dark solid colors (navy, charcoal, black, dark colors)
- Dark photography with sufficient contrast
- Dark gradients
- Brand colors when they provide contrast

**Unacceptable Backgrounds**
- Light backgrounds — use color logo instead
- Busy backgrounds with insufficient contrast
- Transparent backgrounds without defined space

### Contrast Rule

**Minimum Contrast Ratio:** 3:1
- Measure contrast between logo and background
- WCAG AA standard recommends 4.5:1 for text
- Apply 3:1 minimum for logo marks
- Test visibility at intended display size

## Common Logo Errors

### ❌ Do NOT

1. **Stretch or distort** — Logo must maintain proportions
2. **Change colors** — Only use approved color versions
3. **Add effects** — No shadows, glows, strokes, or gradients
4. **Place on high-contrast patterns** — Logo must be clearly visible
5. **Reduce opacity** — Logo should always be opaque and solid
6. **Separate icon from wordmark** — Keep components together
7. **Use outdated versions** — Always use current approved files
8. **Add text or graphics inside** — Logo is locked artwork
9. **Outline or border** — No outline treatment
10. **Rotate or flip** — Orientation must be standard

### ✅ Do

1. **Use approved files** — Download from official sources
2. **Maintain clear space** — Use minimum spacing guidelines
3. **Ensure contrast** — Background must contrast with logo
4. **Scale proportionally** — Keep aspect ratio
5. **Use correct colors** — Reference hex/RGB/CMYK values
6. **Choose appropriate version** — Color for light, reverse for dark
7. **Test visibility** — Verify legibility at intended size
8. **Keep locked** — Logo is finished artwork
9. **Update regularly** — Use current version
10. **Contact for questions** — Reach out for unclear situations

## File Formats & Delivery

### Recommended Formats

| Format | Best For | Notes |
|--------|----------|-------|
| PNG | Digital use, transparency required | Use 300 DPI for print |
| SVG | Web, responsive design, scalability | Resolution-independent, smaller file size |
| PDF | Print, professional documents | Preserves precision, scalable |
| EPS | Professional print, professional design software | Vector format, universal compatibility |
| JPG | Raster use, specific backgrounds | Include padding to prevent cutting |

### File Naming Convention

```
ilsi-europe-logo-[VERSION]-[TYPE]-[COLOR].extension

Examples:
- ilsi-europe-logo-v1-horizontal-color.png
- ilsi-europe-logo-v1-stacked-reverse.svg
- ilsi-europe-logo-v1-icon-only-color.png
- ilsi-europe-logo-v1-horizontal-monochrome.pdf
```

## Digital Applications

### Website

- **Hero/Header:** Primary color logo (100-200px)
- **Favicon:** Icon only (16×16, 32×32, 64×64px)
- **Social Media:** Icon only (400×400px)
- **Footer:** Primary or reverse depending on background

### Social Media

**Profile Picture**
- Format: Square (1:1 ratio)
- Size: 400×400px minimum
- File: PNG with transparent background
- Use: Icon only version

**Header Image**
- Aspect Ratio: 16:9 or platform-specific
- File: PNG or JPG (300 DPI equivalent)
- Use: Horizontal logo variant

### Email

- **Logo Placement:** Top-left or center
- **Size:** 100-150px width
- **Format:** PNG (transparent background)
- **Link:** Logo should link to homepage

### Mobile Apps

- **App Icon:** Icon only, 1024×1024px
- **Splash Screen:** Full logo, centered
- **Navigation:** Icon only (64×64px minimum)
- **Status Bar:** Icon only, if app bar requires it

## Print Applications

### Business Cards

- **Size:** Approximately 1 inch wide (minimum)
- **Position:** Top-left or centered
- **Color:** Reverse (white) on dark background or color on light background
- **Format:** Vector (EPS or PDF)

### Letterhead

- **Position:** Top-left corner or top-center
- **Size:** 0.75–1.5 inches wide
- **Color:** Depending on letterhead background
- **Format:** Vector (EPS or PDF)

### Signage

- **Outdoor/Large:** Ensure 3:1 minimum contrast
- **Minimum Size:** 2–3 inches (small signage)
- **Material:** Follow specification when applying to physical substrates
- **Format:** High-resolution PNG or vector

### Publications

- **Masthead:** 0.5–1 inch
- **Covers:** 1–2 inches
- **Interior Pages:** Footers (0.5–0.75 inches)
- **Resolution:** 300 DPI for print

## Logo Animation (Digital)

### Guidelines for Animated Logos

- **Transitions:** Smooth, professional transitions (200–500ms)
- **Effect:** Subtle entrance animations
- **Speed:** Never jerky or overly fast
- **Looping:** Avoid infinite loops; use purposeful entrance animations

### NOT Recommended

- Continuous rotation or spinning
- Flashing or blinking effects
- Color changes or morphing
- Fragmentation or explosive effects
- Overly long animations (>2 seconds)
